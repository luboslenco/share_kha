# share_kha
Share library for Haxe Kha

## Usage
- Clone repository into 'your_project/Libraries'
- Bring up share dialog(sorry about the naming):
``` hx
share.Share.share(subject:String, body:String, url:String, attachScreenshot:Bool);
```
